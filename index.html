Hello. My name is Mark Finger, and I make web stuff.

You can find me on github or send me some email.

I’m currently floating about Sydney and working on some tech projects that interest me. Some of my time is spent coding at the Interaction Consortium.

A significant portion of my time is spent writing open source projects that solve immediate problems in either my personal or professional environments.

  • unfort
    A build tool for the web that prioritises performance during development and is heavily optimized for hot-swapping code and assets in milliseconds.
  • webpack-build
    A wrapper around webpack that enables persistent caching and compilation within workers processes.
  • python-react
    Pre-rendering React components with Python data.
  • babel-plugin-hot-swap-declarative-modules
    Performs a naive structural scan to identify if a JS module can be hot-swapped without side-effects.
  • record-store
    A data store for a build pipeline that merges asynchronous job requests and handles file changes by intercepting and unrolling in-progress job chains.
  • kv-cache
    Simple persistent-caching for Node. Balances some IO-perf and code-complexity trade-offs by storing records in distinct files.
  • cyclic-dependency-graph
    A JS library for recursively building a cyclic dependency graph. Makes heavy use of immutable data-structures to enable snap-shots to be taken.

I've been active in the open-source world for many years; some of my past projects are still heavily in-use, some were primarily for research or curiosity, and some have been superseded.

  • call-once-after-tick
    A Node library that forces function calls to occur only once and always in the process's next tick.
  • js-host & python-js-host
    Python to Node RPC. Provides a manager process that spawns and controls Node processes, enabling Python processes to seamlessly talk to Node.
  • webpack-yam-plugin & python-webpack-manifest
    A webpack plugin that outputs an asset manifest with relative paths that can consumed by the python library to enable built assets to be integrated into larger systems.
  • optional-django
    Python tools and utils for providing optional support for django's staticfiles, settings, env variables and serializers.
  • python-nodejs & python-npm
    Simple Python wrappers that enable programmatic invocations of the Node and NPM binaries.
  • fatcontroller
    Simple JS pub/sub implementation.
  • scroll-tracker
    A browser library for binding event handlers to element positions that are relative to the viewport.
