Some of the larger open-source projects that I’ve written over the years.

  • unfort
    A build tool for the web that prioritises performance during development and is heavily optimized for hot-swapping code and assets in milliseconds.
  • webpack-build
    A wrapper around webpack that enables persistent caching and compilation within workers processes.
  • python-react
    Pre-rendering React components with Python data.
  • js-host & python-js-host
    Python to Node RPC. Provides a manager process that spawns and controls Node processes, enabling Python processes to seamlessly talk to Node.
